## A. 実験再現用コマンド

本研究の実験は，リポジトリ内のバッチ評価スクリプトにより再現できる。以下は，評価用データセットを入力として，oneshot と sequential を同一条件で実行し，結果を CSV として保存する例である。

```bash
python scripts/evaluate_jp.py \
  --input data/dev_500.csv \
  --output results/results_500.csv \
  --methods sequential oneshot \
  --model gpt-4.1-nano
```

また，結果 CSV から図表を生成する例を示す。

```bash
python scripts/plot_results.py --input results/results_500.csv --outdir results
```
